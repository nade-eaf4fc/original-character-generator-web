# Security Policy

## Supported Versions

This project is currently maintained on the latest `main` branch.

## Reporting a Vulnerability

This repository is a small open-source web project and is not expected to handle high-risk security scenarios, but responsible reports are welcome.

If you find a security-related problem:

1. Please do not post sensitive exploit details in a public issue.
2. Use GitHub private reporting if available.
3. If private reporting is not available, open a minimal public issue and note that you have a security concern.

## Scope Notes

Please report issues such as:

- unsafe handling of imported `settings_json`
- accidental exposure of secrets or personal data
- unsafe requests to local services
- path or storage handling issues

General bugs, UI issues, and feature requests should go through normal GitHub issues.
